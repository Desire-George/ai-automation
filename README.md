# ai-automation
Learning AI automation as a cybersecurity professional, building LLM-powered security tools including alert triage, threat intelligence pipelines and vulnerability analysis agents as part of a 4-week learning plan.

## Projects
- **Week 1** - LLM API integration & security alert classifier
- **Week 2** - RAG pipeline for security policy Q&A
- **Week 3** - CVE analysis agent
- **Week 4** - SOC alert triage tool (full product)

## Stack
Python, Groq API, LangChain, ChromaDB, Streamlit
